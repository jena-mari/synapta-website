## Synapta Website

This is the official website for **Synapta,** the first end-to-end platform that detects AI integrity issues, automates feedback, personalises retention and validates skills for employment. This is a high-performance website built with **React**, **Vite**, and **TypeScript**. This repository contains the source code for the Synapta public ecosystem and showcase.

## üöÄ Overview

This project powers the Synapta web presence, showcasing features, information, and updates about the Synapta ecosystem. It leverages modern frontend tooling to ensure a fast, responsive, and maintainable user experience.

* **Fast Dev:** Powered by **Vite** for near-instant Hot Module Replacement (HMR).
* **Type Safe:** Built with **TypeScript** to catch errors during development.
* **Optimized:** Production-ready builds with tree-shaking and asset optimization.

## üì¶ Getting Started

### 1. Clone the Repository

```bash
git clone https://github.com/jena-mari/synapta-website.git
cd synapta-website

```

### 2. Install Dependencies

```bash
npm install
# or
yarn install

```

### 3. Launch Development Server

```bash
npm run dev

```

The site will be available at `http://localhost:5173`.
